# Comandos — ¿Qué es Git?

En este primer tema no hay comandos de gestión de versiones aún. Sí conviene verificar que Git está instalado correctamente.

---

## Verificar instalación

```bash
git --version
```

Debe mostrar algo como `git version 2.x.x`. Si ves ese mensaje, Git está instalado y listo.

---

## Pedir ayuda directamente desde Git

```bash
git help
```

Muestra una lista de los comandos más comunes. Es útil tenerlo en cuenta: la documentación oficial está siempre a un comando de distancia.

```bash
git help commit
```

Abre la documentación detallada del comando `commit`. Puedes usarlo con cualquier comando.
