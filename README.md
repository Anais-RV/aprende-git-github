# Aprende Git y GitHub

Este es el material de apoyo del curso. Aquí tienes las explicaciones, los comandos de referencia y los ejercicios de cada tema.

No hace falta que sepas nada de Git para empezar. Solo necesitas tener Git instalado, una cuenta en GitHub y ganas de practicar.

Sigue los temas en orden y haz los ejercicios: leer sin practicar no sirve de nada.

---

## Qué vas a aprender

Al terminar este recorrido serás capaz de:

- Entender para qué sirve Git y por qué usarlo
- Gestionar el historial de tu proyecto con commits bien escritos
- Trabajar con ramas y fusionar cambios sin perder el trabajo
- Deshacer errores de forma controlada
- Publicar tu trabajo en GitHub y colaborar con otras personas
- Documentar tu proyecto correctamente

---

## Cómo usar este material

Cada tema tiene su propia carpeta dentro de `temas/` con cuatro archivos:

| Archivo | Qué encontrarás |
|---|---|
| `explicacion.md` | El concepto explicado con contexto y ejemplos |
| `comandos.md` | Referencia rápida de los comandos del tema |
| `ejercicio.md` | Práctica paso a paso para hacer tú mismo |
| `autocheck.md` | Preguntas para comprobar que lo has entendido |

Empieza por `explicacion.md`, prueba los comandos, haz el ejercicio y responde el autocheck. Luego pasa al siguiente tema.

## Convención de commits

Este repositorio usa una convención para escribir los mensajes de commit. Puedes consultarla en [temas/04-commits-bien-escritos/CONVENCIONES.md](temas/04-commits-bien-escritos/CONVENCIONES.md).

## Qué necesitas antes de empezar
