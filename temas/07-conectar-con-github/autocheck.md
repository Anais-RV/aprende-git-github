# Autocheck — Conectar con GitHub

## Preguntas

1. **¿Qué diferencia hay entre Git y GitHub?**
   ¿Podrías usar Git sin GitHub? ¿Y al revés?

2. **¿Qué hace `git remote add origin`?**
   ¿Qué es "origin"? ¿Podría tener otro nombre?

3. **¿Cuál es la diferencia entre `git push` y `git pull`?**
   ¿En qué dirección van los datos en cada caso?

4. **¿Para qué sirve `git fetch`?**
   ¿Qué lo diferencia de `git pull`?

5. **¿Qué hace `git clone`?**
   ¿Qué ventaja tiene respecto a descargar el zip del proyecto?

---

## Comprobación práctica

- ¿Puedes ver tu repositorio en GitHub con todos tus commits?
- ¿Al hacer `git remote -v` aparece la URL de tu repo?
- ¿Los cambios que haces en local llegan a GitHub tras el `push`?
